핵심 변경점

  • 한 페이지에 다 때려넣음 → sub-탭 4개 navigation 도입 — 전체 / 수익 / 건강 / 패턴. 각 sub-탭이 자체 surface, 분석 정보 자연 그루핑
  • 활성 "전체" sub-탭 = 5 state 컴포넌트 (hero / 이번 주 / 카테고리 / 마감 임박 / 오늘 완료). 3초 glance용
  • "수익" sub-탭 = 수익 대시보드 + 월별 트렌드 + 카테고리별 수익
  • "건강" sub-탭 = 라이프 리듬 7일 + 수면 패턴 + 근무 패턴 + 복약 통계 7/30일
  • "패턴" sub-탭 = 생산성 패턴 (시간대/요일 차트) + 카테고리 파이 + 습관 트래커 12주 + 월별 트렌드
  • 액션 탭 redundancy 제거 — dash-next-action 삭제. status bar / Next Action ownership 명확화
  • 모든 이모지 → Lucide inline SVG. sub-탭에도 Lucide 아이콘 (grid / dollar / activity / bar-chart)
