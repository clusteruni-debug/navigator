변경 요지 — sub-tab + 1축 색 + cat-본업 identity 일관

  • Sub-tab nav 5개 (프로젝트 / 캘린더 / 타임라인 / 리포트 / 템플릿) — 4 modal 분산 → 한 탭 내 sub-section. 대시보드 탭 sub-tab 패턴과 cross-tab 일관
  • 1축 stripe 색 (pulse 우선) — 카드 좌측 stripe = deadline pulse 1축만. stale 은 meta 텍스트 "9일 정체" 로 격하 (work-render.js:43-46 의 3축 mix 해소)
  • cat-본업 (indigo) identity — 제목 좌측 3px bar / focus compact 좌측 4px stripe / 새 프로젝트 버튼 soft-tint / sub-tab active. 본업 탭 = 인디고 톤 일관
  • Focus compact 1줄 — expandable 제거. cat-본업 색이 default, 마감 임박 시 --accent-danger 오버라이드. 색이 mode 가 아니라 deadline urgency 를 의미 (axis 단순화)
  • 안정 앵커 4개 상단 — 마감 임박 / 정체 7일+ / 활성 프로젝트 / 이번 주 완료. peripheral scan 단위
  • 헤더 액션 1+1 — primary "+ 새 프로젝트" 단독, "더 보기" 3-dot 메뉴 안에 RO v3 / 템플릿 / 리포트. 4-action bloat 정리. 같은 메뉴 sub-tab 으로도 진입 가능 = 2-way affordance
  • 일반 업무 카드 좌측 3px --cat-본업 — 할일 탭 패턴과 동일 (peripheral scan 일관)
  • 아카이브 격하 — 보류 / 완료 / 보관 = dashed border 하단 collapsed
  • 모든 이모지 → Lucide 인라인 SVG (clock / shield / box / activity / target / calendar / file-text / grid / list-checks / alert-triangle / plus)
  • Hover: bg bg-tertiary + border border-color — Standing rule 준수 (no transform/scale/shadow)
