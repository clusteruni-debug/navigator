변경 요지 — 4-section 시각 위계 + 카테고리 stripe 차별

  • 4-section 명확 분리: 라이프 리듬 → 약 복용 → 습관 → 일상/가족 task (시간순 + 의무 priority 순). uppercase section-title 로 위계 표시
  • 안정 앵커 4개: 오늘 리듬 3/6 / 약 복용 2/3 / 가족 2 / 습관 streak — 매 진입 시 진행률 한 눈에
  • cat-일상 (emerald) identity: 제목 좌측 bar / rhythm hover / habit stripe / 추가 버튼 soft-tint — 본업의 indigo 와 차별
  • 가족 task = cat-가족 (amber) stripe 차별 — 일상 task 와 같은 list 안에 mix 되지만 시각 즉시 구분. resolved IA "일상 ⊃ 가족 inline" 시각 구현
  • filter chip 카테고리 (전체/일상/가족) — task list 내부 필터, navigation axis X. 4-section 안에선 카테고리가 peripheral 만
  • 약 복용 2-group (ADHD / 영양제) — 액션 탭 mockup 01 패턴 차용, cross-tab 일관
  • habit streak inline — 🔥 N 옆에 명시. ADHD dopamine + 완료 토글 우측 정렬 (44pt 터치 타깃)
  • 이모지 → Lucide 인라인 SVG (clock / pill / users / award / sun / leaf / coffee / moon / activity / check / repeat / smile)
  • Hover bg/border only — Standing rule 준수
