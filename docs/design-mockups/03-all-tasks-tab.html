  • task 1개씩 보이면 *전체 분포* 인지 X — 조망 surface가 아니라 *list* surface

핵심 변경점 (v3 final — kanban column 패턴)

  • 2x2 grid kanban — row 1 = 일 (본업/부업), row 2 = 생활 (일상/가족). semantic grouping via position (Bento + Notion sub-grouping 영감). row label 작은 가이드 — 위치 자체가 hierarchy
  • 상단 stable anchors 유지 — 오늘 마감 / inbox / streak (Todoist immutable anchors 패턴)
  • task item compact — 제목 + 마감 chip (오늘 빨강 / 이번 주 노랑 / —). cursor pointer, click → task detail (or 카테고리 섹션)
  • 각 column 상단 = 카테고리 이름 + count + → arrow (섹션 navigate). 카드 click 진입점 패턴 유지
  • 빈 카테고리 (가족) = dashed border-top + "task 없음 + 추가" affordance
  • 이번 주 분포 chart 제거 (사용자 피드백: 필요 없음)
  • 반응형 — PC 4 column / tablet 2 column / mobile 1 column vertical stack
