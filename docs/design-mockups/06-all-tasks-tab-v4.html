v4 변경 요지 — 시간축 primary + 카테고리 = peripheral 시각 도구

  • 시간축 primary 그룹 — 오늘 / 이번 주 / 다음 주 / 나중 / 마감 없음. Things 3 / Linear 표준 패턴 + topology §5.4 resolved IA 적용
  • 카테고리 = peripheral scan — task row 좌측 4px stripe (`--cat-*`) + meta 안 pill (작게 색 + 텍스트). nav axis X, filter chip 으로만 격하
  • 안정 앵커 4개 — 오늘 마감 / 이번 주 / inbox / streak. 매 진입 시 한 눈에 우선순위 파악
  • 이번 주 분포 chart 제거 — task list 자체가 시간 정보 보유, chart 는 cognitive 중복 노이즈
  • 반복 task inline complete — 완료 시 strikethrough + opacity 0.55, 같은 그룹 유지 (다음 instance 자동 등장). resolved IA 적용
  • 액션 호버 노출 — 수정 / 미루기 / 삭제 3 버튼, 평소 시각 노이즈 0. 모바일은 항상 표시
  • D-day chip 색 분리 — <24h 빨강 / <72h 황 / 그 외 회색. 텍스트 안 읽고 색만 보고 긴급도
  • 완료 task = 평면 보존 — 오늘 완료 / 이번 주 완료 collapsed 아카이브 (per resolved IA: 할일 탭 = 진행 중 만, 완료 = 히스토리 탭 cross-cutting)
  • 이모지 X / Lucide 인라인 SVG (clock / calendar / shopping-bag / award / search / repeat / chevron-right / edit / chevrons-right / trash-2 / plus)
  • Hover: bg bg-tertiary + border border-color. transform / scale / shadow X — Standing rule 준수
